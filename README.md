# ExampleHologramPlugin
Depends on the [HologramAPI](https://github.com/MaximFiedler/HologramAPI)

## Commands
Just send these commands in chat
- !spawn
- !bigger
- !smaller
- !normal
- !animate
- !stopanimation
- !kill
- !killall
