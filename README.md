# ExampleHologramPlugin
Depends on the [HologramAPI](https://github.com/MaximFiedler/HologramAPI)

## Showcase video
https://www.youtube.com/watch?v=5GdkMb59naU

## Commands
Just send these messages in chat
- !spawn
- !bigger
- !smaller
- !normal
- !animate
- !stopanimation
- !kill
- !killall
